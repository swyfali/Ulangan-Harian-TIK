Card Game

Project ini dibuat menggunakan Bootstrap 5 dengan konsep permainan kartu sederhana yang menampilkan statistik karakter (seperti di FIFA) menggunakan card, progress bar, dan color utilities. Pada project ini digunakan berbagai komponen Bootstrap untuk membuat tampilan game yang lebih menarik. Selain itu, project ini juga melatih kemampuan dalam mengatur layout website, menggunakan elemen visual modern, serta memahami cara membuat tampilan game berbasis web dengan desain yang lebih profesional dan mudah digunakan. Juga, project ini melatih kemampuan dalam menggunakan css untuk styling atau penampilan dari website.
