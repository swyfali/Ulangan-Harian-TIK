Deskripsi

Nama saya Dismas Harsa, biasa dipanggil Dismas. Saya seorang siswa dari SMA Kolese Kanisius Jakarta, berumur 16 tahun dan memiliki hobi bermain game serta bermain catur. Saya tidak terlalu memiliki pekerjaan impian tertentu, tetapi saya ingin menjadi orang yang sukses dan memiliki kehidupan yang baik di masa depan. Saya sangat senang belajar TIK karena dalam pelajaran ini, saya belajar banyak hal baru mengenai teknologi, terutama tentang HTML, CSS, Bootstrap, dan JavaScript dalam pembuatan website. Pembelajaran TIK membuat saya memahami bagaimana sebuah website dirancang agar terlihat menarik, interaktif, dan responsif di berbagai perangkat. Tujuan saya mempelajari pemrograman web bukan hanya untuk menyelesaikan tugas sekolah, tetapi juga untuk menambah keterampilan yang dapat berguna di masa depan serta membantu saya memahami dunia teknologi yang terus berkembang.
